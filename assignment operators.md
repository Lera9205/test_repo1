# assignment operators
## assignment operators - присваивают значения переменным JavaScript.
**Основные операторы присваивания:**
=    - x = y (тоже самое, что и x = y)
+=   - x += y (тоже самое что и x = x + y)
-=   - x -= y (тоже самое, что и x = x - y)
*=   - x *= y (тоже самое что и 	x = x * y)
/=   - x /= y (тоже самое что и x = x / y)  
%=   - x %= y (тоже самое что и x = x % y)
**=  - x **= y (тоже самое что и x = x ** y)
